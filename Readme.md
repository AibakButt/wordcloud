# Word Cloud Generator

## Description

The Word Cloud Generator is a web application built using React and GraphQL. It generates visually appealing word clouds based on a list of words and their frequencies. The size of each word in the word cloud is determined by its frequency. The application offers features like sorting, filtering, and dynamic word rotation to create captivating and interactive word clouds.

## Features

- Generate word clouds from a list of words and their frequencies.
- Sort words by frequency in ascending or descending order.
- Apply filters to display only relevant words.
- Dynamically rotate words for artistic presentation.

## Getting Started

### Installation

1. Clone the repository:

   ```bash
   cd word-cloud-generator
   cd frontend && npm install 
   cd backend && npm install
    ```

  To start the frontned: ``` npm run dev ```

  To start the backend: ``` npm run start:dev ```


Access the application at http://localhost:3000

### Usage
- Enter words and their frequencies in the input fields.
- Customize sorting and filtering options.
- Click "Generate Word Cloud" to visualize the word cloud.
- Explore and interact with the visually captivating word cloud.

### Technologies
- React: Frontend library for UI development.
- Apollo Client: State management for GraphQL integration.
- GraphQL: Query language and runtime for APIs.
- Tailwind CSS: Utility-first CSS framework for styling.
- Node.js: JavaScript runtime environment.
- Express: Web application framework for Node.js.
- MongoDB: NoSQL database for word data storage.
- Mongoose: MongoDB object modeling for Node.js.
- Redis: In-memory cache for data optimization.
